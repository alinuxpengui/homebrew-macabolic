# Homebrew Tap for Macabolic

This is the official Homebrew tap for [Macabolic](https://github.com/alinuxpengui/Macabolic), a beautiful macOS GUI for yt-dlp.

## Installation

```bash
brew tap alinuxpengui/macabolic
brew install --cask macabolic
```

## Updating

```bash
brew update
brew upgrade --cask macabolic
```

## Uninstalling

```bash
brew uninstall --cask macabolic
brew untap alinuxpengui/macabolic
```

## About Macabolic

Macabolic is a lightweight, native macOS application that provides a beautiful interface for downloading videos from YouTube and other platforms using yt-dlp.

For more information, visit the [main repository](https://github.com/alinuxpengui/Macabolic).
